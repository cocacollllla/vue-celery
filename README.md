### 개발 인원 및 기간

- 개발기간 : 2022/05/24 ~ 2021/05/31
- 개발인원 : 1명

### 적용 기술
- Vue.js
- Firebase
- vuex
- SCSS


### 구현 기능
- firebase로 백엔드서비스를 대신하여 사용
- 카카오 지도 API를 사용해 주소에 따른 지도 노출
- 메뉴를 메인메뉴, 메인토핑, 서브토핑, 드레싱으로 구분하여 칼로리 계산
- 이벤트 종료 날짜 입력하여 이벤트 진행, 종료 구분
- 반응형 구현



## 설치 및 시작 방법
---
```
- git clone https://github.com/cocacollllla/vue-celery.git
- cd vue-celery
- npm install
- npm run serve
```
